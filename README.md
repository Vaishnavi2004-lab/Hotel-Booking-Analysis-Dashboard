# 🏨 Hotel Booking Analysis Dashboard

## 📌 Project Overview

This project focuses on analyzing hotel booking data to identify booking patterns, cancellation trends, customer behavior, and hotel performance. The dashboard was created in Microsoft Excel using Pivot Tables, Pivot Charts, and Slicers to provide interactive insights.

---

## 🎯 Objectives

- Analyze total hotel bookings and cancellations.
- Compare City Hotel and Resort Hotel performance.
- Understand guest type booking behavior.
- Identify monthly booking and cancellation trends.
- Calculate and visualize cancellation rates.

---

## 🛠️ Tools & Technologies Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning & Analysis

## 🔍 Key Insights

- City Hotels receive more bookings than Resort Hotels.
- A significant percentage of bookings are canceled.
- Family and Couple bookings contribute a major share of total bookings.
- Certain months experience higher cancellation rates.
- Booking trends vary based on guest type and hotel category.
