# Hotel Booking Analysis Dashboard

## Project Overview
This project analyzes hotel booking data to identify booking trends, cancellation rates, customer behavior, and hotel performance.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers

## Dataset
- 119,391 hotel booking records

## Key Insights
- Hotel-wise booking analysis
- Guest type analysis
- Monthly cancellation trends
- Booking cancellation rate
